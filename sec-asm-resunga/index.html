Project Overview

This independent security audit was conducted for Resunga Multiple Campus (Tribhuvan University) with explicit authorization. The project focused on identifying vulnerabilities in publicly accessible systems while adhering to strict ethical guidelines and non-intrusive testing methodologies.

Scope of Work

In Scope (Allowed)

  • OSINT & Public Reconnaissance
  • Subdomain Enumeration
  • Non-Intrusive Port Scanning
  • SSL/TLS Configuration Review
  • Security Header Evaluation
  • Technology Fingerprinting
  • Non-Destructive Vulnerability Testing
  • Email Security Checks

Out of Scope (Prohibited)

  • Exploitation of Vulnerabilities
  • Backend/Database Access
  • Password Cracking
  • DDoS or Stress Testing
  • Social Engineering
  • Payload Uploads
  • Any Intrusive Actions
